    public getFaviconsLoaders(params: object): Array source

    Define the list of loaders for the favicons file. The reason this is not with the images loaders is because favicons need to be on the root directory for the browser to automatically detect them, and they only include optimization options for png. This method uses the event reducer webpack-favicons-loaders-configuration-for-browser, which sends the loaders, the received params and expects a loaders list on return.
