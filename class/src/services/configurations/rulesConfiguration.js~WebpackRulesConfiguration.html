    public getFaviconsRules(params: WebpackConfigurationParams): Array source

    Define the list of rules for the favicons file. The reason this is not with the images rules is because favicons need to be on the root directory for the browser to automatically detect them, and they only include optimization options for png. This method uses the reducer event webpack-favicons-rules-configuration-for-browser, which sends the rules, the params and expects a rules list on return.
